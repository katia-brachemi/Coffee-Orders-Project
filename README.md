# 📊 Excel Project : Coffee Sales & Consumption Dashboard (2019–2022)

## Overview

This project was completed by following a guided tutorial from Mo Chen’s Data Analyst YouTube channel, using the dataset provided in the tutorial. 
By reproducing each step, I strengthened my ability to apply diverse Excel formulas and create interactive visuals for a dashboard.

## Objective:  
To analyze and visualize the trends in coffee sales and consumption — Arabica, Excelsa, Liberica, and Robusta — across three countries (UK, Ireland, and USA) between 2019 and 2022.

## 🗂️ Data Source 
Dataset provided by Mo Chen via GitHub.

---

## 📊 Key Skills Practiced

- Applying advanced Excel formulas (SUMIFS, cumulative totals, YTD, MAT)
- Applying advanced formulas such as XLOOKUP and INDEX-MATCH
- Structuring and cleaning data for analysis
- Converting and formatting data for consistency and analysis
- Creating and customizing PivotTables to summarize data
- Using timeline slicers and slicers to filter and interact with visuals
- Connecting multiple visuals together for a cohesive dashboard
- Designing clear, professional charts and visuals
- Building an integrated Excel dashboard to present insights effectively
